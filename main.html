Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM:

  1. document.getElementById(): By this, we can access the IDs declared in the HTML file.
  2. document.getElementsByClassName(): By this, we can access the classes declared in the HTML file.
  3. document.getElementsByTagName(): By this, we can access the elements by tag name in the HTML file, such as the tag.
  4. document.querySelector(): By this, we can select the first element that matches the given element.
  5. document.querySelectorAll(): By this, we can get the list of elements that match the given element.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML contains HTML including text and tags and others while innerText contains only visible texts that are not hidden and textContent contains visible and hidden texts.

Question-3: What is event delegation in the DOM?

Event delegation is a process of handeling child element using parent element like adding evenlistener on parent element instead of adding manually on every child element . It handels the issues like adding child element dynamically and if it needs that evenlistener it automatically gets added to it without the need to manually add it.

Question-4: What is event bubbling in the DOM?

Event bubbling is a mechanism where if an event occurs in the child element ,it then propagates to it's parent, then the parent's parent and this process keeps on going until it reaches the document.

Question-5: How do you create, add, and remove elements using JavaScript?

1.An element can be created using document.createElement()

2.An element can be added using appendChild():appends a child at the end of the parent , append(): appends a list of elements ,prepend() adds a child at the start of the parent element, before():adds an element before that element , after():adds an element after that element.

3.An element can be removed using remove() , a child can be removed from parent using removeChild().
